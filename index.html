Properties CSS Property and Value Definition Example
Color color: ; The color CSS property sets the foreground color value of an element's text and text decorations. Ex: color: dimgray; would make the font that you selected the same color as this page.
RGB rgb: () ;, rgba: () ; The rgb() functional notation expresses a color according to its red, green, and blue components. An optional alpha component represents the color's transparency. Note: CSS Colors Level 4 made some changes to rgb(). Ex: rgb: (105,105,105);, would give you the color "Dimgray" which is the background color of this site.
HSL hsl: ;, hsla: ; HSL stands for hue, saturation, and lightness - and represents a cylindrical-coordinate representation of colors. Ex: hsl: (0,0,41); would set the color to be dimgray which is the background color of this site.
